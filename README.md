# Auth Service Client

Production-ready authentication service client for microservices architecture.

## Features

- ✅ Token verification and validation
- ✅ Login/Logout functionality
- ✅ Token refresh mechanism
- ✅ Menu and permission management
- ✅ Automatic retry with exponential backoff
- ✅ Response caching (Redis/Memory)
- ✅ Comprehensive error handling
- ✅ PSR-3 logging support
- ✅ PSR-16 cache support
- ✅ Ready-to-use middleware
- ✅ Health check endpoint

## Requirements

- PHP >= 8.0
- ext-curl
- ext-json

## Installation

```bash
composer require bpjs/auth-service-client
